# BetterExtensionMod

针对《杀戮尖塔 2》的怪物与事件多样性拓展 Mod，持续更新中。

## 当前进度（v0.1.4）

- 新增多组自定义怪物、遭遇与 Boss 战。
- 新增事件与遗物玩法链路。
- 新增多种自定义 Power 与意图表现。
- 支持多人难度系数配置（`BetterExtension/config/multiplayer_scaling.json`），可按 Act、遭遇、怪物与 Power 细分覆盖。

## 安装方式

1. 从 Releases 下载最新构建产物。
2. 确保以下文件位于同一模组目录下：
   - `BetterExtension.dll`
   - `BetterExtension.pck`
   - `BetterExtension.json`
3. 将该目录放入游戏运行目录下的 `mods/`。
4. 启动游戏，在模组列表中确认 `Better Extension` 已启用。
